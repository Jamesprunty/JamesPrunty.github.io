Linked List

A linked list is a collection of data elements whose order is not given by their physical placement in memory, rather, each element points to the next.
It starts at the head and links to the next element in the array, this then links to the next and so on.
To find data you must go through each element in order.



To start with we have to define the linked list class and create a node function to hold the data.

function LinkedList() {
var length = 0;
var head = null;

var Node = function (element) {
this.element = element;
this.next = null;
};

The "length" variable will keep track of the length of the list.
It also creates a head, which is currently empty.
The node function will take in an element.
It will set "this.element" to the element and it will set the "next" item to null as when we add a new value the one after will be empty.

Add

The add method will add a node onto the end of the list and link it to the previous node.

this.add = function (element) {
var node = new Node(element);
if (head === null) {
head = node;
} else {
var currentNode = head;
while (currentNode.next) {
currentNode = currentNode.next;
}
currentNode.next = node;
}
length++;
};

We start by creating a new node called "node", this will contain the element we want to add.
If the head is null, add the node ass the head.
The currentNode = the head of the list, while there is a next in the list, the current node is equal to the next node.
This will cycle through the list until it reaches the end.
We then set the currentNode.next to be the new node. We will also increase the length by 1.



Remove

The remove method takes in an element to put in, and will remove it from wherever it is in the list.

this.remove = function (element) {
var currentNode = head;
var previousNode;
if (currentNode.element === element) {
head = currentNode.next;
} else {
while (currentNode.element !== element) {
previousNode = currentNode;
currentNode = currentNode.next;
if(currentNode == null){
return false;
}
}
previousNode.next = currentNode.next;
}
length--;
};

We create two nodes here, one for the current node, and one for the previous node. This will mean we can join the previous node and the next node together once it is removed.
If the currentNode is the same as the element, then the head becomes the next element (Which removes the head from the list), the length will minus 1.
While the currentNode is not eqaul to the element we are looking for, move along the list.
If currentNode is null then that means we have hit the end of the list without finding the element so it doesn't exist.
If the element is found then the previous node = the currentNode.next meaning the current node is remove from the list.

IndexOf

The indexOf method takes in an element to find the index of.

this.indexOf = function (element) {
var currentNode = head;
var index = -1;
while (currentNode) {
index++;
if (currentNode.element === element) {
return index;
if(currentNode == null){
return false;
}
}
currentNode = currentNode.next;
}
return -1;
};

We start by setting the "currentNode" to the head, and the index to -1. We start at -1 because the first thing in the loop is to increase the index.
While currentNode (so while it is not null, meaning you haven't reached the end of the list)
If the element is the one we are looking for, return the index.
currentNode = currentNode.next moves the loop along the list and add to the index.
If the currentNode is false return -1 for false.
