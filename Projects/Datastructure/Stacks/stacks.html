Stacks

A stack is like a stack of books, it oporates on a "first in last out" method. An array is an example of this.

The following methods can be used, these methods are already set in the array object.

Push: this will put the element on the top.
Pop: This will take the top element off the top of the stack.
Peek: This will display the top item without taking it off.
Length: This will get the length of the stack.



So let's look at the code here.

//Pop//

this.pop = function() {
if (this.count === 0) {
return undefined;
}
this.count--;
var result = this.storage[this.count];
delete this.storage[this.count];
return result;


The pop method first checks if there is anything in the stack, if not it returns "undefined"

It then removes the count by one. Because the count is now set on the last point of the stack, so the result is equal to the strage value.

This is then deleted from the storage stack which makes the count correct again.
