Sets

A set is similar to stack but has no duplicates and are not in any particular order

The following methods can be used.

Add: This will add a value to the set but only if the set does not already contain the value.
Has: This will return a true or false depending on if the set has the value in it.
Remove: This will remove the value from the set. Values: This will display the values in the set. This is also how you obtain the values of the set.
Size: This will return the set size.
Union: This takes in another set as a parameter, it will merge the sets, only one of each value will be added.
Intersection: This will return all values that are the same in both sets.
Difference: This will return all values that are different in the sets.
subset: This will return a true or false depending if one set found in another set.


So let's look at the code here.

We start off with the collection to hold the set.

var collection = [];

//Has//

this.has = function (element) {
return (collection.indexOf(element) !== -1);
};



The "has()" function takes in an element to check in the set and return a true or false.

If the element is not in the set, the index of the element will equal -1.



//Add//

this.add = function (element) {
if (!this.has(element)) {
collection.push(element);
return true;
}
return false;
};



The "add()" function takes in an element to add to the stack.

If the element is already in the set then it will not add it.

So if the element is not in there, it will push the element onto the collection.


//Values//

this.values = function () {
return collection;
};



The values method returns the collection.


//Remove//

this.remove = function (element) {
if (this.has(element)) {
index = collection.indexOf(element);
collection.splice(index, 1);
return true;
}
return false;
};



First the method checks if the element is in the set.

Then it gets the index of the element.

If it is in there, the collection is spliced at that index to remove it.



//Size//

this.size = function () {
return collection.length;
};



The size method returns the length.



//Union//

this.union = function (otherSet) {
var unionSet = new mySet();
var firstSet = this.values();
var secondSet = otherSet.values();
firstSet.forEach(function (e) {

The Union method takes in another set as its parameter to merge the two together.

The method starts by creating a new set to store the values.

Both set's values are set to a new variable to be able to use.

It goes through each value in the first set and Adds it to the union set. Remember the add() method does not allow for duplicates so we won't get any in our new set.

It does the same for the second set, then returns the unionSet.



//Intersection//

this.intersection = function (otherSet) {
var intersectionSet = new mySet();
var firstSet = this.values();
firstSet.forEach(function (e) {
if (otherSet.has(e)) {
intersectionSet.add(e);
}
});
return intersectionSet;
