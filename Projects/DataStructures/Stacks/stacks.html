Stacks

A stack is like a stack of books, it oporates on a "first in last out" method. An array is an example of this.

The following methods can be used, these methods are already set in the array object.

Push: this will put the element on the top.
Pop: This will take the top element off the top of the stack.
Peek: This will display the top item without taking it off.
Length: This will get the length of the stack.
